# hashnode-assets-backup

Hashnode backup posts to GitHub but assets is linked to cdn.hashnode.com, this action will download the assets and add them to the repo

## Inputs

### `output_path`

## Example usage

```yaml
uses: actions/hashnode-assets-backup@v0.1
with:
  output_path: "assets"
```
